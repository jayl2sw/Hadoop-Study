* JDBC와 호환되는 모든 데이터베이스에 사용 가능

### Sqoop1

* 주요 RDBM 커넥터 지원
* CLI
* RDB to Hive, HBASE 지원



### Sqoop2

* Client가 CLI뿐 아니라 WebUI나 RestAPI, Java API등 다양한 클라이언트 제공
* JDBC 커넥터만 지원
* RDB to Hive, HBase 지원하지 않음
